## Overview

The Workspace Settings page allows you to configure global settings for your Onyx workspace.

<img className="rounded-image" src="/assets/admin/advanced_configs/workspace_settings.png" alt="Workspace settings page" />

### Chat Retention

Setting a Chat Retention policy will delete chat messages after the specified number of days.

<Tip>

  Let your users know what the retention policy is to avoid any surprises!
</Tip>

### Image Processing

Image processing and analysis must be explicitly enabled on this page.

Once enabled, you can set a **Default Vision LLM** for Onyx.

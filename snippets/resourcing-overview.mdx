## Resourcing Overview

To host Onyx, we recommend:

| Resource | Minimum | Preferred |
| --- | --- | --- |
| CPU | 4 vCPU | 8+ vCPU |
| RAM | 10 GB | 16+ GB |
| Disk | 50 GB + ~2.5x indexed data | 500 GB for organizations \<5000 users |

<Warning>
  Vespa does not allow writes once disk usage hits 75%. Make sure to always have some storage headroom.
</Warning>

## Resourcing Overview

| Resource | Minimum | Preferred |
| --- | --- | --- |
| CPU | 4 vCPU | 8+ vCPU |
| RAM | 10 GB | 16+ GB |
| Disk | 50 GB + ~2.5x indexed data | 500 GB for organizations \<5000 users |

<Warning>
  Vespa (the vector database used by Onyx) does not allow writes once disk usage hits 75%.
</Warning>

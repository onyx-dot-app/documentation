# System Description

This page provides a comprehensive overview of Onyx's system architecture and security design.

## Overview

Onyx is designed with security as a foundational principle. Our system architecture ensures that data protection, access control, and privacy are built into every layer of the platform.

## Core Components

- **Frontend Interface**: Secure web application with role-based access controls
- **Backend API**: RESTful API with comprehensive authentication and authorization
- **Vector Database**: Secure storage for document embeddings and metadata
- **LLM Integration**: Secure communication with external language models
- **Connector Framework**: Secure data ingestion from various sources

## Security Principles

- Defense in depth
- Principle of least privilege
- Data encryption at rest and in transit
- Comprehensive audit logging
- Regular security assessments

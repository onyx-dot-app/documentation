Image Generation allows you to create high-quality images using AI-powered models like DALL-E 3.
This feature is available through the Art Action,
which can be equipped to AI Agents to enable image creation capabilities.

## Features

- **AI-Powered Generation**: Create images using state-of-the-art models like DALL-E 3
- **Natural Language Prompts**: Describe what you want to see in plain English
- **High-Quality Output**: Generate professional-quality images suitable for various use cases
- **Multiple Styles**: Create images in different artistic styles and formats
- **Agent Integration**: Use image generation through AI Agents for contextual image creation

## How to Use Image Generation

<Steps>
  <Step title="Select an Agent with Art Action" icon="robot">
    Choose an AI Agent that has the Art Action equipped. The default "Art" Agent comes with this capability,
    or your administrator may have created custom Agents with image generation enabled.
  </Step>

  <Step title="Start a Chat Session" icon="comment">
    Begin a conversation with the Agent that has image generation capabilities.
  </Step>

  <Step title="Request Image Creation" icon="image">
    Ask the Agent to generate an image by describing what you want to see. For example:
    "Create an image of a modern office workspace" or "Generate a logo for a tech startup."
  </Step>

  <Step title="Review and Refine" icon="eye">
    The Agent will generate the image and show you the result. You can ask for modifications, different styles,
    or variations of the generated image.
  </Step>

  <Step title="Download or Use" icon="download">
    Once you're satisfied with the generated image, you can download it or use it in your projects.
  </Step>
</Steps>

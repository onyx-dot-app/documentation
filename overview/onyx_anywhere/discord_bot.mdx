## Overview

The Onyx Discord bot allows you to use Onyx AI chat directly on any Discord server.

Use the official OnyxBot connected to our Cloud, or deploy your own Discord bot with your self-hosted Onyx instance.

<img className="rounded-image" src="/assets/overview/onyx_anywhere/discordbot.png" alt="Discord Bot"/>

## Features

- Install on any server you manage
- Configure behavior per server and channel and custom agents
- Reply or thread-only mode
- Works in forums, threads, and channels

## Try it out

Follow the [installation guide](/admins/miscellaneous/discord_bot)
to install the OnyxBot on your Discord server ([installation
link](https://discord.com/oauth2/authorize?client_id=1460722463431327804)).

To test it out, join the [Onyx community Discord](https://discord.gg/rkCS5RY4)
and tag **@OnyxBot** from from the #support channel!

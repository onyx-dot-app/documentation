Use the Onyx desktop app for a integrated, seamless experience on your computer. Access your organization's knowledge base without opening a browser:

Features include:

- Quick-launch from your system tray or dock
- Global keyboard shortcuts
- Desktop notifications

<img className="rounded-image" src="/assets/overview/onyx_anywhere/desktop_app.png" alt="Onyx Desktop App" />

<Note>
  The Onyx Desktop App provides the same powerful features as the web interface, optimized for desktop use with OS integrations.
</Note>

# Simple Example: Basic Chat Usage

This page demonstrates the most common Chat API workflow - creating a session and sending messages.

## Basic Workflow

1. **Create a chat session** to establish context
2. **Send messages** to interact with the AI
3. **Upload files** if needed for analysis
4. **Delete the session** when finished

## Quick Start Code
```python Python expandable
import requests

API_BASE_URL = "https://cloud.onyx.app/api"
API_KEY = "YOUR_KEY_HERE"

headers = {
    "Authorization": f"Bearer {API_KEY}",
    "Content-Type": "application/json"
}

# 1. Create a chat session
session_response = requests.post(
    f"{API_BASE_URL}/chat/create-chat-session",
    headers=headers,
    json={
        "persona_id": None,
        "description": "Example session"
    }
)

chat_session_id = session_response.json()["chat_session_id"]

# 2. Send a message
message_response = requests.post(
    f"{API_BASE_URL}/chat/send-message-simple-api",
    headers=headers,
    json={
        "message": "Hello! How can you help me today?",
        "chat_session_id": chat_session_id
    }
)
print(message_response.json()["answer"])
```
